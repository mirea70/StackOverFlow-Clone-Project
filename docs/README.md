# :information_desk_person: 팀 인원
<p align="center"> 
  <li> FE : 고영석(팀장)  김태형  엄신영 </li>
  <li> BE : 박성재  박제관 </li>
</p> 
  
  <br/>
  
# 👉 역할
### - BE 개발 담당(기여도 80% 이상)
### - AWS 배포 인프라 구축(EC2, RDS) 담당
### - FE/BE 간 통신 리드
### - 테이블 설계 및 테이블 명세서 작성
### - API 기능 구현
### - 회원 도메인 기능 : 로그인(오류 수정 및 보조), 회원 조회
### - 질문 게시판 기능
  * CRUD
  * 질문 채택/채택 취소
  * 질문 투표 + / -
  * 정렬 기준 옵션
### - 질문에 대한 답변 기능
  * CRUD
  * 답변 투표 + / -
### - 답변에 대한 댓글 기능(CRUD)
### - 검색 기능
### - API 문서화
### - FE/BE 소통 핵심 담당
